# AusteraPlus

A student-grade budgeting API built with **FastAPI**, supporting:
- User authentication (JWT access & refresh tokens)
- Shared family budgets
- Categories and transactions

## Tech Stack
- FastAPI
- SQLAlchemy
- PostgreSQL / SQLite
- JWT Authentication
